JsonAR
======

Plugin JavaScript para consultas em Json

##Documentação
```bash
var data = {
  "Users":[
    {"firstName":"Ray","lastName":"Villalobos","joined":2012},
    {"firstName":"John","lastName":"Jones","joined":2010},
    {"firstName":"Bia","lastName":"Parker","joined":2005}
  ], 
  "Cel": [
    {"num": "(00) 0000-0000"},
    {"num": "(11) 1111-1111"},
    {"num": "(22) 2222-2222"},
    {"num": "(33) 3333-3333"}
  ]
};
```

`JsonAR.main(data);`
